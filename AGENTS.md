## 执行指令

Codex 每次会话启动时，请执行以下步骤：

1. **完整阅读 `CLAUDE.md`**——它是本项目的唯一规范文档，包含：
   - 项目宗旨与架构红线
   - 状态同步铁律
   - 代码规范（文件头注释、函数注释、导入、命名、错误处理）
   - Git 规范（分支策略、提交格式、推送流程）
   - 测试规范
   - 文档维护规则
   - Prompt 管理规范
   - 扩展原则（新增 Agent / Skill / Tool / Graph 节点）

2. **阅读 `AI_CONTEXT.md`** 了解当前项目状态

3. **遵循 `CLAUDE.md` 中的所有规范**，如同它们写在本文件中一样
