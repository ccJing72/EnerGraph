第 2 层
算法模型层 — 9 个纯计算引擎 Mock · 待算法团队就绪
每个模型封装为独立的 MCP Server，通过 JSON Schema 描述输入输出接口。Agent 通过 MCP Client 按需调用，支持热插拔新增模型。
模型自身只做数值计算，不含业务编排和决策逻辑——"算什么"由决策层决定，"怎么算"由模型实现。
